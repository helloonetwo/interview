# 查看项目文件体积

1.安装   webpack-bundle-analyzer
```js
npm install webpack-bundle-analyzer -D
```
2.在 vue.config.js  引入 
```js
module.exports = { 
  chainWebpack: config => { config .plugin('webpack-bundle-analyzer') .use(require('webpack-bundle-analyzer').BundleAnalyzerPlugin) } 
}
```

3.打包 
```js
npm  run  build
```
将得到连接放到浏览器 得到上面的图
